# Files

**Files** - Это программа для поиска удалённых файлов и просмотра всех файлов на пк.

**Как пользоваться?**
1) Нажмите сначала кнопку load и подождите пока загрузится дерево файлов.
2) Нажмите на кнопку старт (треугольник) и ждите пока загрузится дерево удалённых файлов
3) Анализируйте все удалённые файлы
